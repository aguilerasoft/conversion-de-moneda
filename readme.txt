== Description ==


This plugin allows you to do currency conversion in your store.

Where it contains a form that is saved in a database and is reflected in the product.



== Installation == 

1. You must have Woocommerce installed
2. Install and activate the plugin
3. Choose the quantity and the currency you want to show on your products
4. Done!
