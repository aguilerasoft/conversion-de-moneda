== Description ==

This plugin allows you to perform currency conversions in your store. Just add the amount of the currency value along with your currency and it will automatically convert.

Compatible currencies:

== Installation == 

1. You must have Woocommerce installed
2. Install and activate the plugin
3. Choose the quantity and the currency you want to show on your products
4. Done!
